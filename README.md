# AI-Powered Auto Inspection & Repair Assistant

A comprehensive vehicle inspection and repair guidance application powered by AI. Not only diagnose issues, but get detailed step-by-step repair instructions to fix them yourself!

## Features

- **Vehicle Inspections**: Complete digital inspection forms with photo and audio notes
- **OBD-II Diagnostics**: Analyze diagnostic trouble codes (DTCs)
- **AI Repair Guidance**: Get comprehensive DIY repair instructions including:
  - Step-by-step repair procedures
  - Parts lists with cost estimates (OEM & Aftermarket)
  - Required tools and difficulty ratings
  - Safety warnings and precautions
  - Professional repair cost comparisons
- **Repair Resources**: Direct links to parts suppliers and repair tutorial platforms
- **AI Chat Assistant**: Ask questions about repairs, maintenance, and find local services
- **Vehicle History Reports**: Access safety recalls, theft records, and more

View your app in AI Studio: https://ai.studio/apps/drive/1Sbnc1iHk6C-AHfLWRW4ers0aV7TRKwac

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
